The '''_FONT''' function creates a new alphablended font handle from a designated image handle created by {{KW|_NEWIMAGE}} in 32 bit color.



{{PageSyntax}}
:{{Parameter|font_handle}} = {{KW|_FONT}}[({{Parameter|image_handle}})]


{{PageDescription}}
*font_handle is the newly created handle that points to the font.
*image_handle is the handle to the image which you want to retrieve the font from.
