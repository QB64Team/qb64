{{DISPLAYTITLE:_FONT (function)}}
The [[_FONT (function)|_FONT]] function retrieves the font handle from the specified image handle or the current [[_DEST]]ination page's font.


{{PageSyntax}}
:{{Parameter|fontHandle&amp;}} = [[_FONT (function)|_FONT]][({{Parameter|imageHandle&amp;}})]


{{PageDescription}}
*imageHandle&amp; is the handle to the image which you want to retrieve the font from. If not specified, it is assumed to be the current [[_DEST]]ination page.
